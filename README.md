# Photon101

Building small, durable tools for people working in the open.

Current focus:

- **Open-source intelligence**: finding high-leverage issues, maintainers, and projects where useful work compounds.
- **Agent-ready developer tooling**: CLI-first utilities that are easy to inspect, test, and automate.
- **Reproducible research**: plain data pipelines, clear caveats, and no unearned claims.

Active project:

- [`hubsignal`](https://github.com/Photon101/hubsignal): a CLI for surfacing promising open-source issues and repos to work on.

Operating principles:

- Useful before clever.
- Small interfaces, readable code, reproducible outputs.
- No hype, no fake benchmarks, no trading or AI claims without evidence.
